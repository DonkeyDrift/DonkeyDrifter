# IKJL 增量控制快速使用指南

## 🎮 核心特性

**全新 IKJL 控制方式**：模拟"极品飞车"竞速游戏手感
- ✅ 按住加速，松开自动刹车
- ✅ 按住转向，松开自动回中
- ✅ PID 平滑修正，无阶梯感
- ✅ 实时可调参数，自动保存

## 🕹️ 按键说明

| 按键 | 功能 | 行为 |
|------|------|------|
| **I** | 油门增加 | 按住持续加速，松开自动减速至 0 |
| **K** | 刹车/倒车 | 按住持续减速/倒车，松开自动减速至 0 |
| **J** | 左转 | 按住角度持续增加，松开自动回中至 0 |
| **L** | 右转 | 按住角度持续增加，松开自动回中至 0 |
| **空格** | 紧急刹车 | 立即停止车辆 |
| **R** | 录制切换 | 开始/停止数据录制 |
| **M** | 模式切换 | User → Auto Steer → Full Auto |

## 🚀 快速开始

### 1. 启动服务
```bash
cd ~/projects/donkeycar
python manage.py drive
```

### 2. 打开网页
浏览器访问：`http://localhost:8887/drive`

### 3. 启动车辆
点击底部绿色按钮 **"Start Vehicle (space)"**

### 4. 开始驾驶
- **按住 I** → 车辆加速
- **松开 I** → 自动减速至停止
- **按住 J/L** → 左右转向
- **松开 J/L** → 自动回中

## 🎛️ 参数调节

页面右侧滑块可实时调整：

### PID 参数
- **Kp (0.8)**: 响应速度，越大越灵敏（易震荡）
- **Ki (0.0)**: 消除稳态误差，通常保持 0
- **Kd (0.15)**: 阻尼/平滑，越大越稳定（响应慢）

### 速率参数
- **回中速度 (0.35)**: 松开方向键后回中快慢
- **转向角速度 (1.2)**: 按住方向键时转向快慢
- **加速度变化率 (1.0)**: 油门增加/减少速度
- **刹车变化率 (1.2)**: 按 K 键时刹车力度

### 手感预设（建议值）

**街道巡航**（平稳舒适）：
- Kp: 0.6, Kd: 0.2
- 回中: 0.25, 转向: 0.8

**赛道竞速**（灵敏激进，默认）：
- Kp: 0.8, Kd: 0.15
- 回中: 0.35, 转向: 1.2

**越野探险**（稳健耐用）：
- Kp: 0.5, Kd: 0.25
- 回中: 0.2, 转向: 0.6

## 🐛 故障排查

### 按键无反应？

**检查清单**：
1. ✅ 车辆已启动（底部按钮显示 "Stop Vehicle"）
2. ✅ WebSocket 已连接（控制台无 "WS not open" 警告）
3. ✅ 浏览器控制台打开（F12）查看日志
4. ✅ 参数滑块不为 0（回中/转向速度）

**调试日志**（正常时应看到）：
```
IKJL keys active: {up: true, down: false, left: false, right: false}
IKJL sending: {angle: 0.000, throttle: 0.152}
Posting {"angle":0,"throttle":0.152}
```

### 车辆行为异常？

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 转向过猛震荡 | Kp 过大 | 降低 Kp 至 0.5-0.7 |
| 转向响应慢 | Kp 过小 | 提高 Kp 至 0.8-1.0 |
| 回中太慢 | 回中速度过低 | 提高至 0.4-0.6 |
| 回中太快急刹 | 回中速度过高 | 降低至 0.2-0.3 |
| 油门响应慢 | 加速率过低 | 提高至 1.2-1.5 |

### 参数丢失？

参数存储在浏览器 `localStorage`，清除缓存会重置。

**手动恢复默认值**：
1. F12 打开控制台
2. 输入：`localStorage.removeItem('dkc_drive_params')`
3. 刷新页面

## 🎯 使用技巧

### 1. 竞速过弯
```
直道：按住 I 加速
入弯：松开 I，按住 J/L 转向
出弯：角度回中时按 I 加速
```

### 2. 精确停车
```
接近目标：轻点 I（小幅加速）
微调：用 J/L 调整方向
停止：松开所有按键（自动刹车）
```

### 3. 紧急避障
```
发现障碍：按 K 刹车
同时转向：按住 J 或 L
稳定后：松开所有键，自动回正
```

## 🔧 高级功能

### 录制训练数据
1. 按 **R** 开启录制（按钮变橙色）
2. 用 IKJL 正常驾驶
3. 再按 **R** 停止录制
4. 数据保存至 `data/` 目录

### 自动驾驶模式
1. 按 **M** 切换模式：
   - User：完全手动
   - Auto Steer：自动转向，手动油门
   - Full Auto：完全自动
2. IKJL 仍可手动干预

### 导出参数配置
```javascript
// 控制台输入
JSON.stringify(JSON.parse(localStorage.getItem('dkc_drive_params')))

// 复制输出，分享给其他用户
```

## 📝 与原始 IKJL 对比

| 特性 | 原始（步进式） | 新版（增量式） |
|------|---------------|---------------|
| 控制方式 | 每次按键固定步长 | 按住持续变化 |
| 松开行为 | 保持当前值 | **自动回中/减速** |
| 手感 | 阶梯式，需反复按 | **连续平滑 PID** |
| 适用场景 | 精确调试 | **竞速驾驶** |

## ❓ 常见问题

**Q: 可以改回原来的步进式控制吗？**  
A: 原始代码已被替换，若需要可以从 Git 历史恢复或同时支持两种模式。

**Q: 为什么不用方向键？**  
A: 浏览器方向键有默认滚动行为，难以完全阻止，IKJL 更可靠。

**Q: 参数调好后如何导出？**  
A: 参数自动保存在浏览器本地，跨浏览器需手动复制 localStorage。

**Q: 可以用手柄吗？**  
A: 可以，选择 "Gamepad" 模式，IKJL 和手柄可同时使用。
