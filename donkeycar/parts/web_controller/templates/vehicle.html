About Control Modes

Joystick control is provided via the blue touch/click area on screen. Click or touch and drag to control throttle and steering.

Keyboard (IKJL) control with racing game feel:

  • Forward: I (Hold to accelerate, release to auto-brake)
  • Reverse: K (Hold to brake/reverse, release to auto-brake)
  • Left: J (Hold to steer left, release to auto-center)
  • Right: L (Hold to steer right, release to auto-center)
Features: PID smoothing, auto-centering steering, auto-braking throttle. Adjustable via sliders below.

Gamepad control is enabled by the HTML5 gamepad API, currently supported by Chrome and Firefox. Playstation 3 controllers have been confirmed to work.

Device tilt control is enabled for devices with device orientation sensors, and should work with most modern smartphones. Hold your device in landscape mode, tilt forward/backward for throttle and left/right for steering.

Pilot Mode can be toggled or chosen using the keyboard.

  • Toggle Pilot Mode: M
  • User Mode: U
  • Auto Steering Mode: S
  • Full Auto Mode: A

Recording can be toggled using the keyboard.

  • Toggle Recording: R
